Question: how many times method do() would be called?

{% highlight js %} for (i = 0, j=3; i < j ; i++,j-- ) { do(); } {% endhighlight %}

Tip: Try to calculate i and j values for each pass


Answer: 2 times
first pass i = 0 j=3; and i<j is true
second pass i = 1, j=2; and i<j is true
third pass i = 2, j=1; and i<j is false
