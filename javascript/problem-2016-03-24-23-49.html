Question: Develop an snippet to use "if" statement to alert "type is number" if in case the type variable "theVar" is number, "type is string" if type is string less, "type is something else" otherwise.

{% highlight js %} var theVar=15; // "hello", [1]; // put you if here alert("type is number"); alert("type is string"); alert("type is something else"); {% endhighlight %}

Tip: Use if ... else if statement sequence.


Answer:
{% highlight js %} var theVar=15; // "hello", [1]; if (typeof theVar == "number") { alert("type is number"); } else if (typeof theVar == "string") { alert("type is string"); } else { alert("type is something else"); } {% endhighlight %}
